### Achievements in Week 1:

- Understand what AWS is and grasp the basic service categories (Compute, Storage, Networking, Database...).
- Successfully created and configured an AWS Free Tier account.
- Familiarized with the AWS Management Console and know how to find, access, and use services from the web interface.
- Installed and configured AWS CLI on the computer, including Access Key, Secret Key, and default Region.
- Used AWS CLI to perform basic operations such as:
  - Checking account information & configuration.
  - Retrieving the list of regions.
  - Viewing EC2 services.
  - Creating and managing key pairs.
  - Checking running service information.
- Capable of connecting between the web interface and CLI to manage AWS resources concurrently.
